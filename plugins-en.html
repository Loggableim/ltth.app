Extensible Plugin System

Modular architecture with hot-loading and 20+ pre-installed plugins for TikTok LIVE streaming. Includes TTS, Soundboard, Multi-Cam Switching, VRChat integration, RPG crafting, leaderboards and more. Create your own plugins and extend ltth.app infinitely.

Pre-installed Plugins

20+ professional plugins for advanced streaming features

TTS Plugin

75+ TikTok voices & 30+ Google Cloud voices. User-specific voice mapping, automatic TTS for chat messages, blacklist function for specific users or words. Adjustable volume, speed, and pitch settings. Queue management with priority system.

Official Pre-installed

Soundboard Plugin

100,000+ sounds via MyInstants integration. Gift-specific sound mappings with automatic icon display. Event sounds for Follow, Subscribe and Share. Like-Threshold system (e.g., sound at 100, 500, 1000 Likes). Sound picker with search, favorites and trending list.

Official Pre-installed

Multi-Cam Switcher

Control OBS scenes via Gifts or Chat. OBS WebSocket v5 integration. Chat commands like !cam 1-5 for scene switching. Gift-based scene control (e.g., Rose ‚Üí Cam 2). Macro system for complex actions. Cooldown system for spam prevention.

Official Pre-installed

OSC-Bridge Plugin

VRChat avatar control via OSC protocol. Wave gesture on Gifts, Celebrate animation on large donations, Dance trigger, Hearts & Confetti effects. Bidirectional communication with VRChat. Latency under 50ms. Custom OSC parameters supported.

Official Pre-installed

VDO.Ninja Plugin

Multi-guest streaming for interviews and collaborations. Room management with QR code generation, guest management, layout control (Grid, Spotlight, Custom). Direct links for guests, individual audio routing control. 20+ API endpoints.

Official Pre-installed

Topboard Plugin

Leaderboards for top gifters, streaks and donors. Top gifters sorted by total coins, longest gift-streaks tracking, recent donors with timestamp. Real-time updates, own browser source URL for OBS overlay. Customizable styles and rankings.

Official Pre-installed

Emoji Rain Plugin

Animated emoji rain effect for gifts in overlay. Configurable gift-to-emoji mappings (e.g. Rose ‚Üí üåπ, Lion ‚Üí ü¶Å). Adjustable animation speed and emoji density. Integrated into HUD system. Perfect for interactive viewer engagement visualization.

Official Pre-installed

StreamAlchemy Plugin NEW

RPG crafting system with AI-generated items. TikTok gifts transform into virtual RPG items with DALL-E 3 icons. Real-time crafting by combining 2 gifts within 6 seconds. 4 rarity tiers (Bronze, Silver, Gold, Purple). Persistent inventory management. Chat commands: /inventory, /inspect, /merge, /alchemy.

Official Pre-installed

Leaderboard Plugin NEW

Real-time top gifter tracking. Session leaderboard for current stream session and all-time champions. WebSocket-based live updates in overlay. Automatic updates of nicknames and profile pictures. Modern neon/dark theme UI with CSS animations. Performance-optimized with debounced database writes.

Official Pre-installed

Thermal Printer Plugin NEW

Physical event printouts on thermal printers. ESC/POS printer support (USB & network). Prints chat messages, gifts, follows and shares. Smart filtering with min. coins and bot command ignorance. Asynchronous print queue. Auto-reconnect on connection loss. Configurable ESC/POS formatting with ASCII icons.

Official Pre-installed

GCCE Plugin NEW

Global Chat Command Engine. Universal command system for all plugins. Centralized command registry with hierarchical permission system (broadcaster ‚Üí all). Automatic syntax validation and error messaging. Rate limiting (10/user/min, 100 global/min). Unified overlay system. Built-in: /help, /commands, /inventory, /inspect, /merge.

Official Pre-installed

Plugin System Features

Hot-Loading

Load, enable and disable plugins without server restart. Live updates during the stream.

Plugin API

Register callbacks for all TikTok events (Gifts, Chat, Follows, etc.). Event-driven architecture.
