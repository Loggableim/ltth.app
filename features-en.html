Powerful Features for TikTok LIVE Streamers

Discover all features that make PupCid's Little TikTok Helper the ultimate streaming solution for professional TikTok LIVE streamers, agencies, and content creators.

Core Features

TikTok LIVE Integration

Live connection via username without login credentials. Receive Gifts, Comments, Follows, Shares, Likes and Subscriptions in real-time. Auto-reconnect on connection loss. Live statistics and complete gift catalog with all current TikTok gifts.

Text-to-Speech with 75+ Voices

75+ original TikTok voices including German, English, Spanish and many more languages. Optional 30+ Google Cloud voices. User-specific voice mapping, automatic TTS for chat messages, blacklist function, adjustable volume and speed settings.

Alert System

Customizable alerts for all event types (Gifts, Follows, Shares, Likes, Subscriptions). Combination of sound, text and image/GIF. Template system with variables like {username}, {coins}, {message}. Minimum coins filter for premium alerts.

Soundboard: 100,000+ Sounds

Direct access to MyInstants database with over 100,000 sounds. Gift-specific sounds with automatic icon display. Event sounds for Follow, Subscribe and Share. Like-Threshold system (e.g., sound at 100, 500, 1000 Likes). Sound picker with search, favorites and trending sounds.

Goals System

4 separate Goals: Likes, Followers, Subscriptions and Coins. Each goal has its own browser source URL for OBS. Fully customizable styles (colors, fonts, animations, positions). Auto-modes on goal completion: Add (add new goal), Double (double it) or Hide (hide it).

Event Automation (Flows)

If-Then automations without programming knowledge. 6 trigger types (Gift, Follow, Share, Like-Threshold, Subscribe, Chat). 6 action types (TTS, Alert, Sound, Webhook, Write_File, Delay). Complex conditions with 8 operators (==, !=, >, <, >=, <=, contains, regex).

OBS Integration

Transparent Full-HD overlay (1920x1080) as browser source. HUD configuration via Drag & Drop: Event feed, Last Event Spotlight, Top Gifters, Stats and more. Separate browser source URLs for each goal. Event feed shows recent events with icons, usernames and values.

User Profile System

Manage multiple databases for different stream setups. Quick profile switching, complete backup function, separate configurations for gaming streams, IRL streams, special events etc. Each profile has its own settings, flows, soundboard and goals.

Webhook Integration

Send events to external services via HTTP webhooks. Trigger IFTTT, Zapier, Discord, Slack and more. Define custom payloads with event variables. Perfect for advanced automations and multi-platform integrations.

Live Statistics

Real-time stats in the dashboard: Stream Duration, Total Viewers, Peak Viewers, Total Likes, Total Shares, Total Follows, Total Subscriptions, Total Coins. Top gifter leaderboard, gift distribution chart and event timeline.

100% Local & Open Source

All data stays on your computer. No cloud connection required. No login data, no data collection, no tracking. Fully Open Source on GitHub. MIT License. Transparent, secure and community-driven.

Event Automation (Flows)

Create "If-Then" automations without programming. Trigger TTS, Alerts, OBS scenes, OSC actions and more based on TikTok events.

Goals & Progress Bars

4 separate Goals for Likes, Followers, Subscriptions and Coins. Each goal with its own browser source URL for OBS.

Text-to-Speech Settings

75+ TikTok voices and 30+ Google Cloud voices. User voice mappings, Auto-TTS, volume/speed control and blacklist system.

Soundboard Interface

100,000+ sounds via MyInstants. Gift-specific mappings, event sounds, Like-Threshold system and custom uploads.

OBS Studio Integration

Full integration with OBS WebSocket v5 for scene control and browser source overlays
