Installations-Anleitung

1

Setup.exe herunterladen

Klicke auf den Download-Button oben, um die neueste Version von ltth.app für Windows herunterzuladen.

2

Installation ausführen

Öffne die heruntergeladene setup.exe und folge dem Installations-Assistenten. Node.js wird automatisch mitinstalliert falls erforderlich.

3

Mit TikTok verbinden

Starte das Tool, öffne das Dashboard und verbinde dich mit deinem TikTok LIVE Stream. Gib einfach deinen Username ein (ohne @) und du bist startklar!

System-Anforderungen

  • Windows 10 oder neuer (64-bit)
  • Node.js 18.0.0+ (wird automatisch installiert)
  • Aktive Internetverbindung für TikTok LIVE
  • Mindestens 2GB RAM
  • 200MB freier Festplattenspeicher
  • OBS Studio (empfohlen für Overlays)
