Introduction

Welcome to PupCid's Little TikTok Helper documentation. This guide will help you get started and make the most of all available features.

What is ltth.app?

ltth.app is a powerful browser extension designed to enhance your TikTok experience. It provides features like video downloading, ad-blocking, privacy protection, and much more.

Key Features

  • One-click video downloads without watermarks
  • Advanced ad-blocking capabilities
  • Enhanced privacy and security
  • Customizable user interface
  • Plugin system for extensibility

Installation

Installing ltth.app is quick and easy. Follow these steps based on your browser:

Chrome/Edge Installation

  1. Download the extension from the download page
  2. Open your browser's extension page (chrome://extensions or edge://extensions)
  3. Enable "Developer mode" in the top right
  4. Click "Load unpacked" and select the downloaded folder
  5. The extension is now installed and ready to use

Firefox Installation

  1. Download the extension from the download page
  2. Open Firefox and navigate to about:addons
  3. Click the gear icon and select "Install Add-on From File"
  4. Select the downloaded file
  5. Click "Add" when prompted

Quick Start

Get up and running with ltth.app in minutes.

First Time Setup

After installation, ltth.app will automatically activate on TikTok. You'll see a small icon in your browser toolbar indicating the extension is active.

Downloading Your First Video

1. Navigate to any TikTok video
2. Look for the download button added by ltth.app
3. Click the button and choose your preferred quality
4. The video will be downloaded to your default downloads folder

Accessing Settings

Click the ltth.app icon in your browser toolbar to access settings and customize your experience.

Video Downloads

The video download feature allows you to save TikTok videos to your device.

Download Options

  • Quality Selection: Choose from multiple quality options (HD, SD, etc.)
  • Format: Download as MP4 or other supported formats
  • Watermark Removal: Option to remove TikTok watermarks
  • Batch Download: Download multiple videos at once

Usage Example

// Programmatic download (for plugin developers)
ltth.downloadVideo({
    url: 'https://tiktok.com/@user/video/123456',
    quality: 'hd',
    removeWatermark: true
});

API Reference

For developers building plugins or integrations with ltth.app.

Core API

// Get extension version
ltth.version()

// Download video
ltth.downloadVideo(options)

// Get settings
ltth.getSettings()

// Update settings
ltth.updateSettings(newSettings)

// Register plugin
ltth.registerPlugin(plugin)

Events

// Listen for video detection
ltth.on('videoDetected', (video) => {
    console.log('Video detected:', video);
});

// Listen for download completion
ltth.on('downloadComplete', (result) => {
    console.log('Download complete:', result);
});
