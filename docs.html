Architektur

PupCid's Little TikTok Helper basiert auf einer Event-Driven Microservice-Architektur.

  • Backend: Node.js 18-23, Express 4, Socket.io 4
  • Datenbank: SQLite (WAL Mode) mit better-sqlite3
  • TikTok-Integration: tiktok-live-connector 2.1.0
  • OBS-Integration: obs-websocket-js 5.0.6
  • OSC-Protokoll: osc 2.4.5
  • Frontend: Bootstrap 5, Vanilla JavaScript

Module

  • server.js - Haupt-Server mit Express und Socket.io
  • modules/database.js - SQLite-Manager mit WAL-Mode
  • modules/tiktok.js - TikTok LIVE Connector
  • modules/plugin-loader.js - Plugin-System mit Hot-Loading
  • modules/flows.js - Event-Automation Engine

class MyPlugin {
    constructor(api) {
        this.api = api;
    }

    async init() {
        // Express-Route registrieren
        this.api.registerRoute('GET', '/status', (req, res) => {
            res.json({ status: 'ok' });
        });

        // TikTok-Event abonnieren
        this.api.registerTikTokEvent('gift', async (data) => {
            this.api.log(`Gift: ${data.giftName} from ${data.username}`);
        });
    }

    async destroy() {
        this.api.log('Plugin stopped');
    }
}

module.exports = MyPlugin;
