# LTTH Version Archive

This directory stores older versions of PupCid's Little TikTool Helper for archival purposes.

## Purpose

Older stable versions are stored here with the naming pattern:
- `ltth_1.1.1.zip`
- `ltth_1.2.0.zip`
- etc.

## Usage

Users can download previous versions if needed, though the main download always points to `ltth_latest.zip` in the parent directory.

## Note

This directory is maintained for historical reference and compatibility purposes. The launcher application always uses `ltth_latest.zip` for update detection and installation.
