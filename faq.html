Frequently Asked Questions

Find answers to common questions about ltth.app

Ist PupCid's Little TikTok Helper kostenlos?

Ja! Das Tool ist komplett kostenlos und Open Source (MIT-Lizenz). Es gibt keine versteckten Kosten, Premium-Tiers oder Einschränkungen. Alle Features sind für jeden verfügbar.

Muss ich Login-Daten eingeben?

Nein! Du benötigst nur deinen TikTok-Username (ohne @). Keine Passwörter, keine OAuth, keine Login-Daten erforderlich. Alles läuft komplett lokal auf deinem PC.

Welche Betriebssysteme werden unterstützt?

Aktuell wird Windows 10/11 (64-bit) offiziell unterstützt. Linux und macOS sollten funktionieren (mit manueller Installation via npm), sind aber noch nicht offiziell getestet. Native macOS/Linux-Installer sind auf der Roadmap.

Funktioniert das Tool mit OBS?

Ja! Vollständige OBS-Integration mit Browser-Source-Overlays (1920x1080 transparent) und OBS WebSocket v5-Unterstützung für Szenen-Steuerung via Gifts/Chat.

Brauche ich Node.js?

Node.js wird automatisch mit setup.exe installiert, falls nicht vorhanden. Version 18.0.0 - 23.x wird unterstützt. Du musst dich um nichts kümmern.

TikTok-Verbindung schlägt fehl - was tun?

Häufige Lösungen:
• Du musst bereits LIVE sein auf TikTok
• Username muss korrekt sein (ohne @)
• Firewall/Antivirus könnte Verbindung blockieren
• Port 3000 muss verfügbar sein
Mehr Infos: Troubleshooting-Sektion

Kann ich eigene Plugins erstellen?

Ja! Das Plugin-System ist vollständig dokumentiert. Erstelle Plugins mit Express-Routes, Socket.io-Events und TikTok-Event-Callbacks. Siehe Plugin-Dokumentation für Details.

Werden meine Daten in die Cloud hochgeladen?

Nein! Alle Daten bleiben auf deinem Computer. Keine Cloud-Anbindung, keine Datensammlung, keine Tracking. 100% lokal und privat. Optional ist Cloud-Sync für Configs geplant (opt-in).

Wie kann ich Bugs melden oder Features vorschlagen?

Öffne ein Issue auf GitHub: GitHub Issues. Wir freuen uns über Feedback! Das Tool ist in Beta und dein Input hilft bei der Entwicklung.

Verstößt das Tool gegen TikTok's Terms of Service?

Nein. Das Tool nutzt nur öffentlich verfügbare TikTok LIVE-Daten ohne Login. Es ist vergleichbar mit StreamElements, Streamlabs etc. für Twitch. Keine Account-Manipulation, nur Lesen von öffentlichen Events.

Kann ich zum Projekt beitragen?

Ja! Pull Requests sind willkommen. Siehe GitHub Repository für Contribution-Guidelines. Auch Plugin-Entwicklung und Wiki-Dokumentation sind hilfreich!
