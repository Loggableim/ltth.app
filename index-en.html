Why ltth.app for your stream?

Professional streaming features specifically developed for TikTok LIVE

Real-time TikTok LIVE Connection

Connect live to your TikTok stream. Receive Gifts, Comments, Follows, Shares and Likes in real-time without delay. Auto-reconnect on connection loss.

Text-to-Speech System

75+ TikTok voices and 30+ Google Cloud voices. User-specific voice mapping, automatic TTS for chat, blacklist function and customizable volume/speed settings.

Customizable Alert System

Create individual alerts for all event types. Sound + Text + Image/GIF support, template system with variables and minimum coins filter for professional stream presentations.

Soundboard with 100,000+ Sounds

Access to MyInstants database. Gift-specific sounds with icons, event sounds for Follow/Subscribe/Share, Like-Threshold system and sound picker with search and favorites.

Goals & Overlays

4 separate Goals (Likes, Followers, Subs, Coins) with individual browser source overlays. Customizable styles, colors, fonts and auto-modes upon goal completion for your OBS integration.

Event Automation (Flows)

If-Then automations without programming. 6 trigger types, 6 action types, complex conditions with 8 operators. Trigger TTS, Alerts, Sounds, Webhooks and more.

OBS Integration

Transparent Full-HD overlay (1920x1080) for your streams. HUD configuration via Drag & Drop, separate goal overlays and full browser source compatibility.

User Profile System

Multiple databases for different streaming setups. Profile switching, backup function and separate configurations for different stream formats.

100% Local & Secure

All data stays on your computer. No cloud connection, no login required, no data collection. Open Source and completely transparent.

Plugin-System

Modular extension system with hot-loading. 7+ pre-installed plugins (TTS, Soundboard, Multi-Cam, OSC-Bridge, VDO.Ninja, Emoji Rain, Topboard). Create your own plugins with the Plugin API.

VRChat OSC Integration

Control your VRChat avatar via OSC protocol. Wave gesture on Gifts, Celebrate animation, Dance trigger, Hearts & Confetti effects. Custom OSC parameters supported.
