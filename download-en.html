Installation Guide

1

Download ZIP File

Click the download button above to get the latest version of PupCid's Little TikTok Helper.

2

Extract and Start

Windows: Double-click start.exe
Linux: Double-click start-linux or in terminal: ./start-linux
macOS: Double-click start-mac or in terminal: ./start-mac
The tool automatically installs all required dependencies and starts itself.

3

Connect to TikTok

The dashboard opens automatically in your browser. Enter your TikTok username (without @) and connect to your LIVE stream. Make sure you are LIVE!

System Requirements

All Platforms:

  • Node.js 18.0.0 or higher (Download: nodejs.org)
  • Windows 10+ (64-bit), macOS 10.15+, or Linux (Ubuntu 20.04+, Debian 11+)
  • Active internet connection for TikTok LIVE
  • At least 512 MB RAM (1 GB recommended)
  • 200 MB free disk space
  • OBS Studio (recommended for overlays)
